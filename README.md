# Static Website Hosting on AWS (EC2)

## Project Description
This project demonstrates how to host a static website on AWS using an EC2 instance and Apache web server.

## Services Used
- Amazon EC2
- Amazon Linux 2023
- Apache HTTP Server
- Security Groups

## Steps Performed
1. Launched EC2 instance
2. Installed Apache web server
3. Started and enabled HTTP service
4. Deployed static HTML website
5. Accessed website via public IP

## Result
Static website successfully hosted and accessible over the internet.

## Use Case
- Portfolio website
- Landing pages
- Practice AWS fundamentals
